# 🧠 Startup Profit Prediction

This project predicts startup profits using R&D, Marketing, and Admin Spend data.

## 📁 Files in this Repo
- `startup_profit_prediction.ipynb` — Main Jupyter Notebook
- `VC_Startups.csv` — Dataset
- `images/` — Screenshots and plots

## 🔧 Technologies Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-Learn

## 📸 Sample Visual Output
![RD vs Profit](images/scatter_rd.png)

## 🚀 How to Run
1. Clone the repo
2. Open the notebook in Jupyter
3. Run all cells after placing dataset in the same folder
